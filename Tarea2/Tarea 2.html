const nombre_archivo_desde = (título) => título.replace(/[^a-z0-9]/gi, '_').toLowerCase()

Scrapper

Para el scrapper usaremos la librería Playwright. Esta librería es mas completa que simplemente cargar el html: instancia un navegador y funciona tal cual este, esperarando a que se vayan cargando todos los elementos de la página, pudiendo interaccionar con ella: hacer click, rellenar campos de formularios, etc. Además la vamos a utilizar también para testing E2E mas adelante.

Instalamos e inicializamos Playwright: 

$ npm i -D playwright
$ npx playwright install
'-D' es para que se instale como dependencia de desarrollo.

Siguiendo A Guide to Playwright Web Scraping in 2025 y la IA del ejercicio de teoría, hacemos un script scrap-tp.js: 

import {chromium} from "playwright"  // O 'webkit' or 'firefox'
...
// Función auxiliar para crear pausas que simulan el comportamiento humano, asíncrona
const esperar = (mili_segundos) => new Promise(resolve => setTimeout(resolve, mili_segundos));

// También podemos poner esta función sin usar Promises
async function espera(ms) {

const browser = await chromium.launch({ headless: true });   // headless: sin abrir el navegador, más rápido
await browser.newContext({                                   // simula el chrome de escritorio
   userAgent: 'Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/145.0.0.0 Safari/537.3'
})

const page = await browser.newPage()
try {
	// para que salgan todos los productos de Tienda Prado
	await page.goto('https://tiendaprado.com/es/385-impresiones?resultsPerPage=999', {timeout:8000});
} catch (error) {
	console.error(error)
	process.exit(1)
}

page.once('load', () => console.log('Página cargada'))
await page.waitForTimeout(3000);  // espera 3 segundos

Para recuperar la información que nos interese, es mejor usar los locators de playwright, ya tienen espera autmática y son más robustos que otro métodos anteriores. Para acceder a el/los elemento(s) que nos interesa(n), se puede usar la misma sintáxis de los selectores de css, p.e. 

const locators_páginas = page.locator('.thumbnail-container > a')

locator responde o bien con una lista si encuentra varios elementos, o bien con un objeto con métodos similares a la API del DOM si encuentra uno solo getAttribute(), innerText() o textContent() entre otros: 

const lista_páginas = []
for (const loc of await locators_páginas.all()) {   // son varios elementos encontrados
  const página = await loc.getAttribute('href')     // un solo elemento ahora
  lista_páginas.push(página)
}
