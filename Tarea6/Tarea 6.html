Token de autentificación en una cookie

Para mantener la autentificación entre distintas páginas usaremos Json Web Tokens, que se intercambian entre el servidor y el navegador en una How to securely use JSON Web Tokens in a cookie with Node.js.

Cuando se comprueban las credenciales, se emite un token JWT con la identificación del usuario y en su caso los permisos: 

// ./routes/usuarios.ts
import jwt from "jsonwebtoken"

router.get('/login', (req, res) => {
	res.render('login.njk', {error:false})
})

router.post('/login', async (req, res) => {
  // desde el formulario
  const {email, contraseña} = req.body 

  try {
    const usuario = await prisma.usuario.autentifica(email, contraseña)
    const token = jwt.sign({usuario: usuario.nombre, admin:usario:admin}, process.env.SECRET_KEY)

    res.cookie("access_token", token, {
        httpOnly: true,
        secure: process.env.IN === 'production'
      }).redirect('/')

  } catch(error) {
    logger.error(error.message)
    res.render('login.njk', {error:true}		
  }
})				

router.get('/logout', (req, res)=> {
  // resetear la cookie 
  // redirigir a '/'
})
Middleware de autentificación

En un middleware se comprueba el token y se pone la información en el request y en locals, para que esté accesible en adelante: 

// index.ts 
import cookieParser from 'cookie-parser'
import jwt from 'jsonwebtoken'
...
app.use(cookieParser())
...

// middleware de autentificación, justo antes de los routes
// para que solamente les afecte a estas rutas
app.use((req, res, next) => {
  const token = req.cookies.access_token;
  if (token) {
     const data = jwt.verify(token, process.env.SECRET_KEY);
     req.usuario = data.usuario                      // username 
     req.admin   = data.admin                        // y role en el request
     app.locals.usuario = data.usuario               // y accesible en las plantillas
     app.locals.admin   = 
     logger.info(`Autentificado ${data.usuario} admin:${data.admin}`)                         
  } else {
     app.locals.usuario = undefined
     app.locals.admin = undefined
  }
  next()
})
