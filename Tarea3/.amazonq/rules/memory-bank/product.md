# Product Overview

## Project Purpose
Database management system for products using PostgreSQL with Prisma ORM. This project demonstrates modern database interaction patterns using TypeScript, containerized PostgreSQL, and type-safe database queries through Prisma.

## Key Features
- **PostgreSQL Database**: Containerized PostgreSQL database using Docker Compose with Alpine Linux for minimal footprint
- **Prisma ORM**: Type-safe database client with automatic TypeScript type generation
- **Full-Text Search**: PostgreSQL full-text search capabilities for product queries
- **Database Migrations**: Version-controlled schema changes with Prisma Migrate
- **Data Seeding**: Automated database population from JSON data files
- **Visual Database Management**: Prisma Studio for GUI-based data inspection and editing

## Core Capabilities
- Create and manage product catalog with título, descripción, precio, and imagen fields
- Execute type-safe database queries using generated Prisma Client
- Perform full-text searches across product descriptions and titles
- Seed database with initial product data from JSON files
- Migrate database schema changes with version control
- Inspect and modify data through Prisma Studio web interface

## Target Users
- Developers learning modern ORM patterns with TypeScript
- Students studying database management and containerization
- Backend developers implementing product catalog systems
- Teams requiring type-safe database interactions

## Use Cases
- Product catalog management for e-commerce applications
- Learning PostgreSQL with modern TypeScript tooling
- Prototyping database schemas with migration support
- Demonstrating containerized database development workflows
